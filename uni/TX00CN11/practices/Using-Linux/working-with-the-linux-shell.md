# Working with the Linux Shell

```bash
ls # non hidden files

ls -a # displays all files including hidden ones

# .bash_history (Stores the command we used in the bash, when we use the up arrow key)

history # list of all command that has been entered

clear # clears the screen

## Executing files

## 1 type the entire directory of the file
/home/raghib/helloworld # helloworld

## 2 Using tilde
~/helloworld

## 3 Exists in the current directory
./helloworld

## 4 Move the current directory to the environment path variable for eg. the bin dir
## copy the file to bin dir and then type
helloworld

```
